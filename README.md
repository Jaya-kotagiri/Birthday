This project is an interactive pixel-animation web experience that leads the user to a personalised birthday video. The website presents a small playful “catch the cat” interaction where animated characters guide the user through a short story before revealing the final surprise.

I enjoy building personal projects like this alongside my academic work because I learn best by creating and experimenting. Projects like this allow me to explore new ideas in animation, interaction design, and storytelling while strengthening my technical skills through hands-on implementation.

Beyond the technical aspects, this project reflects skills that are important to me as a developer: creativity, attention to detail, user-centric thinking, and the ability to translate an idea into an engaging experience. From designing the interaction flow to implementing the animations and transitions, the goal was to build something playful, memorable, and meaningful.
